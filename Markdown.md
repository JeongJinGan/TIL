## 코드 블록

### fenced code block

- `(backtick) 기호 3개를 활용하여 작성.
- 특정 언어를 명시하면 Syntax highlighting 기능이 적용된다.

```html
print('hello')
# 주석
<h1>
    제목
</h1>
<!-- 주석 -->
```

``` python
print('hello')
# 주석
```

위 같은 코드 블록은 실행은 되지 않고 단순히 메모.
